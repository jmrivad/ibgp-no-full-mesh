# Supporting Material for *draft-rivadeneyra-no-ibgp-full-mesh*

This repository contains the supporting material for the Internet-Draft:

**"No iBGP Full-Mesh in Stub Autonomous Systems"**  
Author: JM Rivadeneyra  
Datatracker page: https://datatracker.ietf.org/doc/draft-rivadeneyra-no-ibgp-full-mesh/

---

## 📘 Purpose of this Repository

The goal of this repository is to provide **clean, high‑quality SVG versions** of all figures referenced in the Internet-Draft.  
Because the IETF Datatracker and `xml2rfc` do not currently render inline SVG images, the draft includes the SVGs as text inside `<artwork>` blocks.  

This repository allows reviewers and implementers to:

- View the figures properly rendered  
- Download them individually  
- Reuse them for analysis or discussion  
- Track changes across draft versions  

---

## 📝 Notes for Reviewers

- The figures in the draft appear as **raw SVG text** due to current limitations of `xml2rfc` and the Datatracker.
